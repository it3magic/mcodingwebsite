# Deployment Workflow

## ⚠️ IMPORTANT

**This GitHub repository is connected to Netlify and is the source of truth for the live site.**

- **Live Site:** https://m-coding.ie
- **GitHub Repo:** https://github.com/it3magic/mcodingwebsite
- **Platform:** Netlify (auto-deploys on push to main)

## Workflow for Updates

1. Make changes in THIS repository (`mcodingwebsite/`)
2. Commit and push to main branch
3. Netlify automatically deploys (1-2 minutes)
4. Verify changes at https://m-coding.ie

## Important Notes

- Local directories (`m-coding/`, `m-coding-git/`) are NOT deployed
- Only changes pushed to this GitHub repo will appear live
- Always update the GitHub repo for production changes
